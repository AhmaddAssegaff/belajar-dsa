# Belajar DSA (Data Structures & Algorithms)

Repository ini berisi catatan dan implementasi latihan **Data Structures & Algorithms (DSA)** menggunakan berbagai pendekatan dan studi kasus sederhana.

Tujuan repository ini:
- Melatih problem solving
- Memahami struktur data dan algoritma dasar
- Membiasakan diri dengan optimisasi solusi
- Persiapan technical interview dan competitive programming dasar

---

# Requirement Dasar

Sebelum belajar DSA, disarankan sudah memahami dasar pemrograman seperti:

- Variabel
- Tipe data
- Function / Method
- Looping (`for`, `while`)
- Conditional (`if`, `else`)
- Array / List dasar
